Glance is a language that can be used to help find objects on screen

When possible using on screen information as a guide to your desired element

In our first Example we put a dynamic label, each level the label contains a glance selector pointing to the element you should click on.

Sometimes the Description is not enough by itself.

In these cases we want to add context to our selectors

the > sign is used to narrow down our scope

If we want to click on the inner square "circle > square" would be instructing the selector to for a square that is in the scope of a circle
